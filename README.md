# Blinkit_Dashboard_PowerBI

## Overview

This project presents an interactive **Power BI dashboard** built using a Blinkit grocery sales dataset. The dashboard provides comprehensive insights into sales performance, customer behavior, product categories, and outlet operations through dynamic and visually appealing reports.

## Objectives

* Analyze overall sales performance.
* Identify top-performing product categories and items.
* Compare outlet performance by location, size, and type.
* Track customer ratings and purchasing trends.
* Enable data-driven decision-making through interactive visualizations.

## Key Features

* Interactive slicers and filters
* KPI cards for Total Sales, Average Sales, Number of Items, and Average Rating
* Sales analysis by product category and fat content
* Outlet performance by establishment year, size, and location
* Dynamic charts and drill-down analysis
* Clean, user-friendly dashboard design

## Tools & Technologies

* Power BI
* Power Query
* DAX (Data Analysis Expressions)
* Microsoft Excel

## Dashboard Insights

* Evaluated overall business performance using key KPIs.
* Identified the highest-selling product categories and outlets.
* Analyzed customer ratings across different outlet types.
* Compared sales trends across outlet sizes and geographic locations.
* Created interactive reports to support business decision-making.

This project demonstrates practical skills in **data cleaning, data modeling, DAX, data visualization, and business intelligence** using Power BI.
